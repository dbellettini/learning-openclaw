# Contributing

This is a personal learning project, not a collaborative effort — so there is no roadmap, no issue tracker, and no expectation that pull requests will be reviewed promptly (or at all).

That said, help is always welcome. If you spot something broken, have a suggestion, or want to share a useful trick you found while learning OpenClaw, feel free to open an issue or a PR. No guarantees, but it will be appreciated.

For contributions to OpenClaw itself, head to the upstream repo: https://github.com/openclaw/openclaw
